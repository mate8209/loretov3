# Loreto, Corrientes — Proyecto de Ordenamiento Territorial

Presentación web para GitHub Pages del Trabajo Final 2026 del Doctorado en Desarrollo Territorial (UNRC).

## Contenido

- `index.html`: presentación principal.
- `.nojekyll`: evita problemas de publicación en GitHub Pages.
- `img/`: imágenes y láminas documentales usadas en la presentación.
- `docs/guion_oral.html`: guion de apoyo para la exposición.

## Cómo publicar

1. Subir todos los archivos de esta carpeta a la raíz del repositorio.
2. Verificar que `index.html` quede en la raíz.
3. Activar GitHub Pages: `Settings → Pages → Deploy from branch → main → /root`.
4. Esperar unos minutos y abrir la URL generada.

## Navegación

- Flecha derecha: siguiente slide.
- Flecha izquierda: slide anterior.
- Botón `Guion`: abre el guion oral.
